# Template Sécurité Python

TPs sécurité Python ESGI 4A

## Installation

```bash
poetry lock
poetry install
```

## TP1 - IDS Network Analyzer

```bash
poetry run tp1 -i eth0 -c 100 -o rapport.pdf
```

## TP2 - Shellcode Analyzer

```bash
poetry run tp2 -f shellcode_easy.txt -o rapport.pdf
poetry run tp2 --all
```
